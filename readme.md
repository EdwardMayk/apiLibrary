# API LIBRARY


## Installation

Instalar las dependencias del proyecto

```bash
npm i
```

Ejecutar la DB y Redis

```bash
docker compose up
```

## Ejecutar los siguientes comandos

```bash
npm run build
```
## Modo de desarrollo
```bash
npm run dev
```

o

```bash
npm run start
```
